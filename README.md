# Realistic Earth Globe Explorer Pro

Realistic Earth Globe Explorer Pro is an interactive frontend world visualization project created with HTML, CSS, and JavaScript.

The project displays a realistic Earth-style globe with blue ocean colors, green and brown land areas, moving cloud layers, atmosphere glow, country labels, city markers, animated connection lines, and clickable information panels.

---

## Project Purpose

The purpose of this project is to create a realistic and interactive world globe interface for portfolio, GitHub, and LinkedIn presentation.

The project is designed as an Earth explorer system. Users can rotate the globe with the mouse, zoom in to reveal countries and cities, and click a country or city to display detailed information.

This project is stronger than a basic map page because it includes animation, coordinate projection, interactive zoom levels, location data, and dynamic UI panels.

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html`.
4. Drag the globe with the mouse to rotate it.
5. Scroll to zoom in or zoom out.
6. Click a country label to see country information.
7. Click a city marker to see city information.

No backend, database, framework, API key, or installation is required.

---

## Interaction Logic

The project uses latitude and longitude data for countries and cities.

At different zoom levels, the interface shows different details:

* Low zoom: Earth, ocean, continents, clouds, and atmosphere
* Medium zoom: country labels
* High zoom: city markers and city names
* Click interaction: information panel opens for the selected country or city

The globe is rendered on Canvas using mathematical projection logic. Country and city positions are calculated from longitude and latitude coordinates and projected onto the rotating globe.

---

## Visual Details

### Ocean and Sea Design

The ocean uses multiple blue gradients to create a deeper and more realistic Earth appearance.

### Land Areas

Continents are drawn with green and brown tones to separate land from water clearly.

### Cloud Layer

Moving cloud shapes are rendered above the globe to create a more realistic Earth effect.

### Atmosphere Glow

A blue atmosphere glow surrounds the planet to make it look more like Earth from space.

### Country and City Layers

Country names appear when the user zooms in. City markers and city names appear at deeper zoom levels.

### Information Panel

When the user clicks a country or city, the panel displays related information such as capital, region, population, latitude, and longitude.
