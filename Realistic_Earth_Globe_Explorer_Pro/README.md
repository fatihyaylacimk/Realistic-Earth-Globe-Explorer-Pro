# Realistic Earth Globe Explorer Pro

Realistic Earth Globe Explorer Pro is an interactive frontend world visualization project created with HTML, CSS and JavaScript.

The project displays a realistic Earth-style globe with blue ocean colors, green/brown land areas, cloud layers, atmosphere glow, country labels, city markers and clickable information panels.

## Project Purpose

The purpose of this project is to create a more realistic world globe interface for portfolio, GitHub and LinkedIn presentation.

The project is designed as an interactive Earth explorer. Users can rotate the globe with the mouse, zoom in to reveal countries and cities, and click a country or city to display information.

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html`.
4. Drag the globe with the mouse to rotate it.
5. Scroll to zoom in or zoom out.
6. Click a country label to see country information.
7. Click a city marker to see city information.

No backend, database, framework or installation is required.

## Interaction Logic

The project uses latitude and longitude data for countries and cities.

At different zoom levels, the interface shows different details:

- Low zoom: Earth, ocean, continents and atmosphere
- Medium zoom: country labels
- High zoom: city markers and city names
- Click interaction: information panel opens for selected country or city
